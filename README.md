# NYC Planning GIS Team

This is the primary repository for the GIS team at the NYC Department of City Planning (DCP). [Other GIS Team repos exist](https://github.com/NYCPlanning/?q=gis+OR+fast-tracker&type=all&language=&sort=), but will be migrated into this central repository over time.

This repo is very much a work in progress.

Questions and concerns can be directed to [DCPOpenData@planning.nyc.gov](mailto:dcpopendata@planning.nyc.gov).
